Learn more about palettes

Each palette is built from a base color using harmony rules (analogous, complementary, etc.) and variations in lightness/saturation to get practical combinations.

Harmonies guide the relationship between colors: for example, complementary creates contrast, analogous generates smooth cohesion, and monochromatic works with variations of a single hue.

Use the darkest color for text and backgrounds, the most vivid for calls to action, and the intermediate ones for accents and secondary backgrounds. Test contrast and accessibility before applying.
